FAQ

Can I connect using another client?

Yes! This chat client uses IRC behind the scenes and you can also connect using a number of third-party chat clients with the configuration settings below:

  • Server: irc.freenode.net
  • Channel: #officenomads

What is freenode?

From their FAQ: Freenode is a special-purpose, not a general-purpose, discussion network, currently implemented on Internet Relay Chat (IRC). It exists to support specific communities. It provides an interactive environment for coordination and support of peer-directed projects, including those relating to free software and open source. Our aim is to help our participants to improve their communicative and collaborative skills and to maintain a friendly, efficient environment for project coordination and technical support. For more information about the network philosophy, please take a look here.

What's with the Captcha?!?

Since we use freenode, which is a public chat service, they do now allow autoconnect so that bad people can't abuse the system.
